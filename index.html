Pmodp wire format

El header de un mensaje Pmodp: 


version-pair      msg-type              checksum (xxhash32)
+-----------------+---------------------+------------------+
| 1 octet         | 1 octet             | 4 octet          |
+-----------------+---------------------+------------------+

+                 +                     +
|                 |                     |
v                 v                     v

version   adapter    kind         code
+-------+-------+    +-----+---------+
| | | | | | | | |    | | | | | | | | |
+-------+-------+    +-----+---------+
4-bit     4-bit      3-bit     5-bit
